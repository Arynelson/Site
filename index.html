Escape Room

Escape Room Game is a simple and interactive puzzle adventure game where the player must explore different rooms, find hidden items, and unlock the secret 4-digit code to escape.

Developed with Python and Tkinter, this game features a user-friendly graphical interface (GUI) with clickable buttons for exploration and interactions.

You can play it by downloading the Main.py file from GitHub and running it in your code editor.

Job Search

Job Finder is a web application designed to help users search for job opportunities in Germany, save jobs, track applications, and manage their hiring process efficiently. The platform integrates with the Arbeitnow API for real-time job listings and provides secure user authentication for personalized job tracking.

Tech Stack

  • Frontend: React, Vite, HTML, CSS, TailwindCSS, Axios, JavaScript
  • Backend: Node.js, Express.js, Bcrypt, PostgreSQL
  • Authentication: JWT (JSON Web Tokens)
